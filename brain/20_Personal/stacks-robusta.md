# Stack Tecnológica

Estas são as stacks de preferência, organizadas por categoria e maturidade de uso.

Stack utlizada para criação de projetos robusto, projetos com ideias ja validadas, projetos que precisao de escalabilidade, segurança e robustez

## Stack Robusta

- **Front-end**: Angular, Tailwind CSS, Typescript
- **Back-end**: Java, Springboot

## Banco de Dados
- **Relacional:** PostgreSQL.
- **NoSQL:** MongoDB, Redis (Cache).

## Infraestrutura & Cloud
- **Containers:** Docker, Docker Swarm (Orquestração).
- **Cloud Providers:** AWS, Digital Ocean.
- **Web Servers/Proxy:** Nginx, Traefik.

## DevOps & CI/CD
- **Automação:** Jenkins (Jenkinsfile).
- **Monitoramento:** Prometheus, Grafana.
- **Qualidade de Código:** SonarQube.

## Testes
- **E2E:** Playwright.
- **Unitários:** JUnit (Java), Jest/Supertest (Node.js).

## Documentação & API
- **Modelagem:** Swagger/OpenAPI.
- **API Client:** Postman.

## Versionamento & Gestão de Código
- **Git Vendors:**:  Bitbucket.
- **Workflow:**: GitFlow.
