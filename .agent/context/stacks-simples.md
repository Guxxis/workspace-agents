# Stack Tecnológica

Estas são as stacks de preferência, organizadas por categoria e maturidade de uso.

Stack utlizada para criação de projetos simples, validação de ideias e criação de MVPs, visando velocidade de criação e resolução de problema temporarios

## Stack Simples

- **Front-end**: React, Tailwind CSS, Typescript
- **Back-end**: Node.js, Express

## Banco de Dados
- **Relacional:** PostgreSQL.

## Infraestrutura & Cloud
- **Containers:**: Docker, Docker Swarm (Orquestração).
- **Cloud Providers:**: Digital Ocean.
- **Web Servers/Proxy:**: Traefik.

## Testes
- **E2E:** Playwright.
- **Unitários:** Jest/Supertest (Node.js).

## Documentação & API
- **Modelagem:** Swagger.
- **API Client:** Postman.

## Versionamento & Gestão de Código
- **Git Vendors:**:  Bitbucket.
- **Workflow:**: GitFlow.
