# Stack Tecnológica

Estas são as stacks de preferência, organizadas por categoria e maturidade de uso.

## Front-end
- **Frameworks:** Angular (Enterprise), React (Next.js para Web Apps).
- **Styling:** Tailwind CSS, Vanilla CSS (Modern), Bootstrap (Legado/Admin).
- **Linguagem:** TypeScript (Sempre).
- **CMS:** WordPress (Projetos rápidos/Landing Pages).

## Back-end
- **Java:** Ecossistema Spring (Boot, Data, Security, Cloud).
- **Node.js:** Express, NestJS (TypeScript).
- **PHP:** WordPress Development.

## Mobile
- **Cross-Platform:** Flutter (Preferencial), React Native.
- **Native:** Android (Kotlin).

## Banco de Dados
- **Relacional:** PostgreSQL (Preferencial), MySQL.
- **NoSQL:** MongoDB, Redis (Cache).

## Infraestrutura & Cloud
- **Containers:** Docker, Docker Swarm (Orquestração).
- **Cloud Providers:** AWS, Digital Ocean.
- **Web Servers/Proxy:** Nginx, Traefik.

## DevOps & CI/CD
- **Automação:** Jenkins (Jenkinsfile), GitHub Actions.
- **Monitoramento:** Prometheus, Grafana.
- **Qualidade de Código:** SonarQube.

## Testes
- **E2E:** Playwright.
- **Unitários:** JUnit (Java), Jest/Supertest (Node.js).

## Documentação & API
- **Modelagem:** Swagger/OpenAPI.
- **API Client:** Postman, Insomnia.

## Versionamento & Gestão de Código
- **Git Vendors:** GitHub, GitLab, Bitbucket.
- **Workflow:** GitFlow ou TRUNK based development.
